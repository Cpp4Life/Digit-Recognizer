=========== Update the python version to the latest ===========
- Option 1: Open Command Prompt (cmd)
   Step 1: type 'where python'.
   Step 2: grab the directory to 'C:\Users\asus\AppData\Local\Programs\Python\Python37\' by Ctrl + C.
   Step 3: type 'cd C:\Users\asus\AppData\Local\Programs\Python\Python37\' and Enter.
   Step 4: type the command 'python -m pip install --upgrade' and Enter. --> Done.

- Option 2: Run cmd as administrator (This only works when you've already included the path to C drive since installing python.)
   type the command 'python -m pip install --upgrade' and Enter. --> Done.

=========== Install packages ===========
- Option 1: Open Command Prompt (cmd)
   Step 1: type 'cd C:\Users\asus\AppData\Local\Programs\Python\Python37\Scripts' and Enter.
   Step 2: just simply type the name of package you want following this structure 'pip install *package*'.
   Ex: install numpy --> 'pip install numpy'

- Option 2: Run cmd as administrator
   type package's name and run it 'pip install *package*'.
   Ex: install numpy --> 'pip install numpy'
